# 🎂 Birthday Surprise Website

A beautiful interactive birthday surprise website featuring a real-time countdown, custom lock screen, animated birthday reveal, photo gallery, heartfelt messages, and special effects designed to create a memorable birthday experience.

## ✨ Features

* 🔒 Custom Lock Screen
* ⏰ Real-Time Countdown Timer
* 🎉 Automatic Birthday Reveal
* ❤️ Personalized Birthday Message
* 📸 Photo Gallery Section
* ✨ Smooth Animations & Transitions
* 📱 Fully Responsive Design
* 🎵 Optional Background Music Support
* 🎁 Interactive Surprise Experience

## 🎯 How It Works

1. User opens the website.
2. A custom lock screen appears.
3. Countdown runs until the birthday time.
4. When the target date and time arrive, the surprise unlocks automatically.
5. Birthday wishes, animations, and memories are revealed.
